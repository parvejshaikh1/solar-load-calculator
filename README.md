# Solar Load Calculator (AI Powered)

##  Overview
AI-powered system that extracts electricity bill data (PDF/Image) using OCR and automatically generates Excel reports with solar size and savings estimation.

##  Features
Upload Electricity Bill (PDF/Image)
OCR-based data extraction
Automatic Excel generation
Solar size calculation
Savings estimation

##  Tech Stack
Python
Flask
Tesseract OCR
OpenCV
Pandas
OpenPyXL

##   Workflow
1. Upload bill
2. AI extracts data
3. System fills Excel
4. Download final report

---

## Output Example
Consumer Number
Units Consumed
Bill Amount
Load (kW)
Solar Size
Estimated Savings

---

##  Use Case
Used in solar companies to reduce manual bill analysis time from 20 minutes to seconds.
