# ☀️ Solar Load Calculator Automation

## 📌 Overview
This project automates electricity bill analysis for solar load calculation using AI-based OCR. It extracts data from electricity bills (PDF/Image) and automatically generates an Excel report with solar system size and savings estimation.

---

## ⚡ Features
- Upload electricity bill (PDF/Image)
- Extract important bill data automatically using OCR
- Fill Excel template automatically
- Calculate solar system size
- Estimate electricity savings
- Download generated Excel output

---

## 🧠 Technologies Used
- Python
- Flask
- Tesseract OCR
- OpenCV
- pdfplumber
- openpyxl
- HTML / CSS

---

## 📂 Supported File Types
- PDF
- JPG
- JPEG
- PNG

---

## 🔄 Workflow
1. User uploads electricity bill
2. OCR extracts required data
3. System processes and calculates solar requirements
4. Excel file is generated automatically
5. User downloads final report

---

## 📊 Output Fields
- Consumer Number
- Units Consumed
- Bill Amount
- Load (kW)
- Solar Size (kW)
- Estimated Savings

---

## 🎯 Use Case
Used in solar companies to reduce manual electricity bill analysis time from 15–20 minutes to a few seconds using automation.

---

## 🔥 Project Impact
Reduces manual workload and improves accuracy in solar system planning using AI automation.
